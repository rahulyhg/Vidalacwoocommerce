== Description ==

WPC Product Bundles is a plugin help you bundle a few products, offer them at a discount and watch the sales go up!

= Live demo =

Click to see [live demo](https://demo.wpclever.net/woosb/ "live demo")

= Features =

<ul>
<li>Search & pick products at a glance</li>
<li>Drag & drop to re-arrange</li>
<li>Set quantity for each product</li>
<li>Set min/max quantity for a bundled product or whole bundle</li>
<li>Auto calculate the regular & sale price</li>
<li>Show bundled products above add to cart button or in a new tab</li>
<li>Ability to add the variable product or exactly a variation to the bundle</li>
<li>Mark the bundle is customizable (change the quantity of bundled products)</li>
<li>Mark the bundle is virtual or downloadable</li>
<li>WPML integration</li>
</ul>

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Product Bundles", Install & Activate it
4. Now when you create/edit a product you can choose product type is "Smart Bundle", then add products as you want
5. Almost done! click Save and see the result
